# 🏠 House Price Prediction: Regression Modeling

![Scikit-Learn](https://img.shields.io) ![Machine Learning](https://img.shields.io)

## 📌 Project Overview
Built a machine learning pipeline to predict residential home prices using regression techniques. This project focuses on data cleaning, handling outliers, and optimizing model accuracy.

## 🛠️ Technical Workflow
* **Data Cleaning:** Handled missing values in 15+ categorical features.
* **Feature Engineering:** Log-transformed skewed price data to improve model normality.
* **Modeling:** Compared Linear Regression, Ridge, and Lasso to minimize RMSE.
